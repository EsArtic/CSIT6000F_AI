This program is implement by python 3.6 and also use third partly library numpy for computation.
Before running the program, please make sure that there are no version conflicts or module missing.

After running the program, a new file containing prediction output will be generate.
The weights map and the accuracy will be print on the screen directly.

If you are using Linux/Mac OS, please use run.sh to run the program.
If you are using Windows OS, please use run.bat to run the program.
The program take two command line arguments as inputs:
-- The first one is the trainning set file path, default_value = "./training-set.csv"
-- The second one is the output file path, default_value = "./output.csv"
